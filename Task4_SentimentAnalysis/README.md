# Task 4 - Sentiment Analysis

## Objective
Perform sentiment analysis on customer reviews using Natural Language Processing (NLP) and visualize the results.

## Dataset
Women's Clothing E-Commerce Reviews Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- TextBlob

## Steps Performed
- Loaded the dataset
- Cleaned missing review text
- Performed sentiment analysis using TextBlob
- Classified reviews as Positive, Negative, or Neutral
- Created visualizations
- Generated business insights

## Visualizations
- Sentiment Distribution
- Sentiment Pie Chart
- Rating Distribution
- Average Rating by Sentiment
- Recommendation Distribution

## Business Insights
- Most reviews are Positive.
- Average customer rating is above 4.
- Around 82% of customers recommend the products.
- Very few Neutral reviews.

## Outcome
Successfully analyzed customer sentiment and generated useful business insights using NLP.
