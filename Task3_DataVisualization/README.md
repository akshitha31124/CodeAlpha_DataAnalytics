# Netflix Data Visualization

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python. It includes data cleaning, exploratory data analysis (EDA), and multiple visualizations to gain insights into Netflix content.

## Dataset
Netflix Movies and TV Shows Dataset

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
- Movies vs TV Shows
- Top 10 Countries
- Ratings Distribution
- Release Year Distribution
- Top Genres
- Netflix Content Added by Year
- Movie Duration Distribution
- Top Directors
- Missing Values Heatmap
